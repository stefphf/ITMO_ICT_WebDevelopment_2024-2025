Задание 2. Фильтрация данных

Текст задания

По созданным в пр.1 данным написать следующие запросы на фильтрацию:

  • Где это необходимо, добавьте related_name к полям модели
  • Выведете все машины марки “Toyota” (или любой другой марки, которая у вас есть)
  • Найти всех водителей с именем “Олег” (или любым другим именем на ваше усмотрение)
  • Взяв любого случайного владельца получить его id, и по этому id получить экземпляр удостоверения в виде объекта модели (можно в 2 запроса)
  • Вывести всех владельцев красных машин (или любого другого цвета, который у вас присутствует)
  • Найти всех владельцев, чей год владения машиной начинается с 2010 (или любой другой год, который присутствует у вас в базе)

Задание 3. Агрегация и аннотация

Текст задания

Необходимо реализовать следующие запросы c применением описанных методов:

  • Вывод даты выдачи самого старшего водительского удостоверения
  • Укажите самую позднюю дату владения машиной, имеющую какую-то из существующих моделей в вашей базе
  • Выведите количество машин для каждого водителя
  • Подсчитайте количество машин каждой марки
  • Отсортируйте всех автовладельцев по дате выдачи удостоверения (Примечание: чтобы не выводить несколько раз одни и те же записи воспользуйтесь методом .distinct())

Самая ранняя дата выдачи удостоверения

from django.db.models import Min
earliest_license = DriverLicense.objects.aggregate(Min('issue_date'))
print("Самая ранняя дата выдачи удостоверения:", earliest_license['issue_date__min'])

Вывод:

Самая ранняя дата выдачи удостоверения: 2014-03-05

Самая поздняя дата владения машиной

from django.db.models import Max
latest_ownership = Ownership.objects.aggregate(Max('start_date'))
print("Самая поздняя дата владения машиной:", latest_ownership['start_date__max'])

Вывод:

Самая поздняя дата владения машиной: 2024-11-04

Количество машин у каждого владельца

from django.db.models import Count
users_with_cars = User.objects.annotate(num_cars=Count('ownership__car')).values('id', 'username', 'num_cars')
for user in users_with_cars:
    print(f"ID: {user['id']}, Username: {user['username']}, Машин: {user['num_cars']}")

Отсортировать всех автовладельцев по дате выдачи удостоверения

sorted_owners = User.objects.filter(driverlicense__isnull=False).order_by('driverlicense__issue_date').distinct()
for owner in sorted_owners:
    license = owner.driverlicense_set.first()
    print(f"ID: {owner.id}, Username: {owner.username}, License Date: {license.issue_date}")
